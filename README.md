📊 Inventory Management System (Excel MIS Project)

🚀 Project Overview

This project is a fully functional Inventory Management System built using Microsoft Excel.
It demonstrates how Excel can be used as a powerful tool for MIS reporting, automation, and decision-making.

---

❌ Problem

Managing inventory manually can lead to:

 Stock mismanagement
 Delayed reordering
 Data entry errors

---

✅ Solution

I developed an Excel-based system that automates inventory tracking and provides real-time insights.

---

🔧 Features

 📦 Stock Tracking (Stock In / Stock Out / Current Stock)
 ⚠️ Automated Low-Stock Alerts
 🏭 Supplier Management with dropdown selection
 📊 Interactive Dashboard with KPIs
 🤖 Automation using formulas and VBA
